Fundamental Concepts of HTML

HTML (HyperText Markup Language) is the standard language used to create and design webpages. It provides the basic structure for web content by using elements and tags.

1. HTML Elements and Tags

HTML documents are made up of elements, which are represented by tags. Tags are enclosed in angle brackets (< >), and most have an opening tag and a closing tag.

2. Basic Structure of an HTML Document

<!DOCTYPE html>
<html>
<head>
    <title>My First Web Page</title>
</head>
<body>
    <h1>Welcome to HTML</h1>
    <p>This is a basic paragraph.</p>
</body>
</html>

3. Common HTML Elements

Some commonly used elements in HTML include:

  • Headings: <h1> to <h6>
  • Paragraphs: <p>
  • Links: <a href="https://example.com">Click here</a>
  • Images: <img src="image.jpg" alt="Description">
  • Lists:
    • Ordered List: 
      <ol>
    <li>Item 1</li>
    <li>Item 2</li>
</ol>
    • Unordered List: 
      <ul>
    <li>Item A</li>
    <li>Item B</li>
</ul>

4. Attributes

HTML elements can have attributes that provide additional information. Example:

<a href="https://example.com" target="_blank">Visit Example</a>

5. Forms and Input Elements

Forms allow user input through various input fields:

<form>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <input type="submit" value="Submit">
</form>

6. Semantic HTML

Semantic elements help in defining the meaning of web content, improving accessibility and SEO.

  • <header>
  • <footer>
  • <article>
  • <section>

7. Multimedia Elements

HTML supports embedding multimedia:

  • Audio: <audio>
  • Video: <video>
  • Embedding a webpage: <iframe>

8. CSS in HTML

CSS (Cascading Style Sheets) is used to style HTML documents. Examples:

<style>
    body { background-color: lightgray; font-family: Arial, sans-serif; }
    h1 { color: darkblue; }
</style>

9. JavaScript in HTML

JavaScript is used for interactivity in web pages. Examples:

<script>
    function greet() {
        alert("Welcome to my website!");
    }
</script>
<button onclick="greet()">Greet</button>

Understanding these fundamental concepts will help you build well-structured and interactive web pages using HTML, CSS, and JavaScript.
