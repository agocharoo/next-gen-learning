CSS Fundamentals

What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages.

Basic CSS Syntax

selector { property: value; }

Beginner Level CSS

  • Selectors and Properties
    • p { color: red; }
  • Colors and Backgrounds
    • body { background-color: lightblue; }
  • Font Styling
    • h1 { font-family: Arial, sans-serif; }
  • Box Model
    • div { margin: 10px; padding: 15px; border: 2px solid black; }

Intermediate Level CSS

  • Flexbox Layout
    • .container { display: flex; justify-content: center; }
  • Grid Layout
    • .grid-container { display: grid; grid-template-columns: 1fr 1fr; }
  • Positioning Elements
    • .box { position: absolute; top: 50px; left: 100px; }
  • Transitions and Animations
    • .button { transition: background-color 0.3s ease; }

Advanced Level CSS

  • CSS Variables
    • :root { --main-color: blue; } p { color: var(--main-color); }
  • Advanced Selectors
    • p:first-child { font-weight: bold; }
  • Responsive Design
    • @media (max-width: 600px) { body { background-color: grey; } }
  • CSS Frameworks

    • Popular CSS frameworks include Bootstrap, Tailwind CSS, and Foundation.

Example

The following CSS styles a paragraph:

p { color: blue; font-size: 16px; }
