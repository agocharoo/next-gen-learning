CSS Notes

1. What is CSS?

CSS is a stylesheet language used to describe the presentation of an HTML document. It controls the layout, colors, fonts, spacing, and responsiveness of web pages.

2. How to Apply CSS

There are three main ways to apply CSS to an HTML document:

A. Inline CSS

Adding CSS directly within an HTML element using the style attribute.

<p style="color: blue; font-size: 20px;">This is a blue paragraph.</p>

Use case: Quick styling but not recommended for large projects.

B. Internal CSS

Writing CSS inside a <style> tag in the <head> section of an HTML document.

<head>
  <style>
    p {
      color: blue;
      font-size: 20px;
    }
  </style>
</head>

Use case: Useful for single-page styling but not ideal for large projects.

C. External CSS (Recommended)

Writing CSS in a separate .css file and linking it in HTML.

<head>
  <link rel="stylesheet" href="styles.css">
</head>

styles.css

p {
  color: blue;
  font-size: 20px;
}

Use case: Best for scalability and maintainability.

3. CSS Selectors

Selectors define which HTML elements the styles apply to.

A. Universal Selector (*)

* {
  margin: 0;
  padding: 0;
}

B. Element Selector

h1 {
  color: red;
}

C. Class Selector (.)

.red-text {
  color: red;
}

Example:

<p class="red-text">This text is red.</p>

D. ID Selector (#)

#main-title {
  font-size: 24px;
}

Example:

<h1 id="main-title">Main Title</h1>

❌ Avoid using IDs for styling; use classes instead.

E. Grouping Selector (,)

h1, p, a {
  color: blue;
}

F. Descendant Selector (space)

div p {
  color: green;
}

G. Pseudo-classes

a:hover {
  color: red;
}

4. CSS Properties

CSS consists of properties (what you want to change) and values (how you want to change it).

A. Text Styling

h1 {
  font-size: 30px;               /* Size */
  font-family: Arial, sans-serif; /* Font */
  color: purple;                 /* Text color */
  text-align: center;            /* Alignment */
  text-transform: uppercase;     /* Capitalization */
}

B. Backgrounds

body {
  background-color: lightgray;         /* Solid color */
  background-image: url('image.jpg');    /* Image */
  background-size: cover;
}

C. Borders & Padding

div {
  border: 2px solid black;  /* Thickness, style, and color */
  padding: 10px;            /* Space inside */
  margin: 20px;             /* Space outside */
}

5. The Box Model

Every HTML element is a rectangular box with:

  • Content – The actual text or image.
  • Padding – Space inside the border.
  • Border – The edge around the padding.
  • Margin – Space outside the border.

Example:

.box {
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 5px solid black;
  margin: 30px;
}

6. Flexbox (For Layout)

Flexbox helps create responsive layouts.

.container {
  display: flex;
  justify-content: space-between; /* Distribute items */
  align-items: center;            /* Align items vertically */
}

7. Media Queries (For Responsive Design)

Used to adjust styles based on screen size.

@media (max-width: 600px) {
  body {
    background-color: lightblue;
  }
}

8. CSS Grid (Advanced Layout)

Used for complex layouts.

.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* Three columns */
  gap: 10px;
}
